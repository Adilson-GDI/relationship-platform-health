# Decisions

Registro inicial de decisoes tecnicas do projeto.

## 2026-07-05 - Estrutura base do repositorio

Decisao: organizar o repositorio em `app/`, `platform/`, `packages/`, `docs/` e `tools/`.

Motivo: preparar a Relationship Platform Health para evoluir de um app inicial, o FitCheck, para uma base reutilizavel de aplicativos de saude e bem-estar.

Impacto: nenhuma regra de negocio foi alterada nesta decisao. A extracao de codigo para `packages/` fica para uma etapa futura, quando houver mais seguranca sobre os limites de reutilizacao.
